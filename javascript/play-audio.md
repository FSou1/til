# How to play an audio file

In order to play an audio file, you can use the `Audio` type:

```javascript
const url = 'https://interactive-examples.mdn.mozilla.net/media/cc0-audio/t-rex-roar.mp3';
const myAudio = new Audio(url);
myAudio.play();
```
